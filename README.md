# 画板

基于 Leaferjs 开发的画板

UI 仿照 excalidraw

[预览地址](https://luminous.xjq.icu)

## 功能

- 基础能力

  - [x] 画布拖拽、缩放
  - [x] 图形编辑
    - [x] 移动
    - [x] 缩放

- 基础图形

  - [x] 矩形
  - [x] 圆
  - [x] 线
  - [x] 文字
  - [] 画笔
  - [] 图片

- 导出能力

  - [x] 背景设置
  - [x] 导出图片(png, jpg)
  - [x] 数据导出

- [x] 撤销重做
- [] 设置器

## 开发

node > 18

pnpm === 9

安装依赖

```sh
pnpm i
```

启动

```sh
pnpm dev
```
